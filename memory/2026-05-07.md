Victor tested `/new` with Neo after creating `BOOTSTRAP.md` for the Neo workspace. Same behavior as Ceil: no greeting delivered after `/new`.

**Pattern confirmed:**
- Ceil: `/new` → system ack ✅ → no greeting ❌ → requires follow-up message
- Neo: `/new` → system ack ✅ → no greeting ❌ → requires follow-up message
