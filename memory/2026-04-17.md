### Key Design Decisions
- **Single `.env` file** — Docker Compose auto-reads `.env` from project directory
- **Minimal required config** — Users only need to set `N8N_PASSWORD`
- **Auto-generation for everything else** — `RAILS_MASTER_KEY`, `SECRET_KEY_BASE`, database passwords
- **Volume persistence** — Credentials survive container restarts via `prod_credentials` volume

### User Flow (Production)
```bash
git clone https://github.com/ceil-labs/agent-platform.git
cd agent-platform
cp .env.example .env
# Edit N8N_PASSWORD
docker compose -f docker-compose.ghcr.yml up -d
```
