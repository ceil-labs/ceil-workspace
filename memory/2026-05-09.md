**Three architectural approaches identified:**
1. **Sandboxed iframe + raw HTML** â€” Fastest to implement, self-contained, secure. Best for one-off reports.
2. **React component registry + JSON config** â€” Full React hooks, can call Memoria APIs, type-safe. Best for dashboards/tuners.
3. **Preact runtime in iframe** â€” ~10KB self-contained shareables. Best for S3 uploads/exports.

**Recommendation:** Start hybrid with iframe + `htmlBody` column on `wiki_articles` table. Fastest path. Graduate to component registry when artifacts need live Memoria data.
