**Issue 1 — Citations appearing as 0:**

- Fix: createArticle now returns `citationCount` + `parentSlug`; setCitations returns full counts

**Issue 2 — parentSlug null in list:**
- Root cause: list query only selected `parentId` (UUID), not `parentSlug`
- Fix: LEFT JOIN on parent table, both `parentId` and `parentSlug` returned

**Issue 3 — Bad chunkIds silently failing:**
- Root cause: FK violation on citation insert would fail silently or roll back article
- Fix: Pre-validation before any embedding/insert work; clear 400 with missing IDs listed

## Late Session Updates

### Memoria Cross-Reference System (New Feature)

**New endpoints for article-to-article links:**
- `POST /api/wiki/:slug/links` — create links (mode: replace/append)
- `PATCH /api/wiki/:slug/links/:linkId` — update a link
- `DELETE /api/wiki/:slug/links/:linkId` — remove a link

**Link types:**
- `prerequisite` — must read this first
- `supersedes` — this article replaces the older one
- `see-also` — related topic
- `references` — default, general reference

**Links appear on article detail:** `GET /api/wiki/:slug` now returns `links` (outgoing) and `backlinks` (incoming).
