```
Browser → Tailscale → Nginx → auth_request /api/session/check
                              ↓
                           Rails returns 200 or 401
                              ↓
                           200 → proxy to n8n:5678
                           401 → redirect to /session/new
```

   - Added `check` to `allow_unauthenticated_access` list so nginx can reach it
