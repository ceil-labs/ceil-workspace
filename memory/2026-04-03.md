**Goal:** Enable production deployment via pre-built Docker images for easier distribution.

**Created Files:**

| File | Purpose |
|------|---------|
| `.github/workflows/ghcr.yml` | Builds and pushes to GHCR on every push to master |
| `docker-compose.ghcr.yml` | Production compose that pulls pre-built image |
| `DEPLOYMENT.md` | Quick start guide for users/deployers |

**GHCR Workflow Features:**
- Triggers on push to `master` or tags `v*`
- Multi-platform builds with Docker Buildx
- Caching via GitHub Actions cache
- Image tags: `master`, semver, and SHA

**Deployment Options Now Available:**

| Method | Use Case | Command |
|--------|----------|---------|
| **GHCR (Recommended)** | Users deploying production | `docker compose -f docker-compose.ghcr.yml up -d` |
| **Build Locally** | Custom builds/development | `docker compose -f docker-compose.prod.yml up --build` |
| **Development** | Local development with hot-reload | `docker compose up` |
