# 2026-04-02 — Daily Log

## Memory System Migration

### Problem
- OpenClaw responses were slow (~60s delays)
- Logs showed `Failed to save messages to Honcho: TimeoutError`
- `memorySearch.multimodal` config error blocking startup

### Investigation
- Honcho memory plugin was timing out on every message sync
- QMD memory initialization failing due to multimodal embedding requirement
- `memory-core` plugin was disabled

### Fixes Applied

1. **Uninstalled Honcho plugin**
   ```bash
   openclaw plugins uninstall openclaw-honcho
   ```

2. **Disabled multimodal memory** (was causing provider adapter errors)
   - Set `agents.defaults.memorySearch.multimodal.enabled: false`
   - Set `agents.list[1].memorySearch.multimodal.enabled: false` (Neo agent)

3. **Enabled memory-core plugin**
   ```bash
   openclaw config set plugins.entries.memory-core.enabled true
   ```

4. **Verified slot configuration**
   - `plugins.slots.memory: "memory-core"` ✓
